# Tegshe Car Rental System
The Rental Car Management System is designed to simplify and automate the vehicle rental process for customers and rental companies. It enables customers to search, reserve, rent, and return vehicles, while allowing administrators to efficiently manage vehicle inventory and reservations. The system improves operational efficiency, prevents booking conflicts, and enhances the overall customer experience through a scalable and object-oriented software design.


# Glossary

1. **Reservation**
   An agreement made by a customer to rent a selected vehicle for a specified rental period.

2. **Customer**
   A registered user who can browse available vehicles, make reservations, rent cars, and manage their bookings.

3. **Admin**
   An authorized company employee responsible for managing vehicles, reservations, and other system operations, including adding, updating, and removing vehicles.

4. **Car Rental System**
   A software platform that enables customers to search for, reserve, rent, and return vehicles while allowing administrators to manage rental operations.

5. **Cancellation Fee**
   A fee charged when a customer cancels a reservation within 24 hours of the scheduled pickup time.
